# AI Prompts Used During Development

This file contains some of the basic prompts used to get help from AI while building the **Cine-Stream Movie Explorer** project.

---

## 1. React Project Setup

Prompt:
"How to create a React project using Vite and organize folders for components, pages, and context?"

Purpose:
To understand the basic project structure and setup for the React application.

---

## 2. Fetch Movies from API

Prompt:
"How to fetch movie data from an API in React using useEffect and display it in a component?"

Purpose:
To load movies from the API and display them on the home page.

---

## 3. Movie Card Component

Prompt:
"Create a reusable React component for displaying movie cards with poster, title, and a favorite button."

Purpose:
To display each movie in a structured card layout.

---

## 4. Favorites Feature

Prompt:
"How to implement a Favorites feature in React using Context API and localStorage?"

Purpose:
To allow users to add and remove movies from their favorites list.

---

## 5. React Router Navigation

Prompt:
"How to implement page navigation in React using React Router for Home and Favorites pages?"

Purpose:
To create a multi-page experience without reloading the page.

---

## 6. Mood Based Movie Recommendation

Prompt:
"Return ONLY a single movie title based on a given mood using the Gemini API."

Purpose:
To generate AI-based movie recommendations based on the user's mood.

---

## 7. UI Improvements

Prompt:
"How to style a movie grid layout using CSS or Tailwind for a responsive design?"

Purpose:
To improve the visual appearance of the application.

---

## 8. GitHub Deployment

Prompt:
"How to push a React project to GitHub using git commands?"

Purpose:
To upload the project to GitHub for version control and sharing.

---

These prompts were used only for guidance during development while implementing the Cine-Stream project.
